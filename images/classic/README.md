# Classic action icons

These SVGs preserve Mapper's established action-icon vocabulary while allowing
Qt to render it cleanly at the requested display scale. The corresponding PNGs
in `images/` are the visual references; the SVGs use compact semantic shapes,
paths, strokes, and gradients rather than embedded bitmaps.

The alternative Material drawings remain in `images/material/` for possible
future use. `resources.qrc` currently selects this classic set.
