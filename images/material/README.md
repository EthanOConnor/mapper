# Material Symbols

The SVG files in this directory are the rounded 24 px variants selected from
the Google Material Design Icons repository at commit
`819d78680a849ceef4c78f863d8753e3160b7c89`.

Source: https://github.com/google/material-design-icons

They are redistributed unmodified under the Apache License 2.0 in
`LICENSE.txt`. Mapper maps these source drawings to its semantic action names
through `resources.qrc`; the source filenames are intentionally descriptive
rather than tied to a widget or pixel size.
